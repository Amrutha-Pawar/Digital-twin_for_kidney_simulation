# 🧠 AI Powered Kidney Digital Twin Simulation Platform

An advanced full-stack healthcare simulation platform that creates a **Digital Twin of the Human Kidney** to analyze patient health conditions, predict kidney stress/damage, simulate treatment outcomes, and generate AI-assisted medical reasoning reports.

This project combines:

* 🧪 Medical parameter simulation
* 🤖 AI-driven reasoning engine
* 📊 Interactive health analytics dashboards
* 🫀 Kidney condition visualization
* 📄 Automated medical report generation

---

# 🚀 Project Overview

The **Kidney Digital Twin** is a virtual computational model that mimics the condition of a patient's kidney using biomedical inputs such as:

* Creatinine
* Blood Urea
* GFR (Glomerular Filtration Rate)
* Blood Pressure
* Sugar Levels
* Hydration Status
* Protein Leakage
* Medication Response

Based on these health parameters, the system:

✔ predicts kidney condition severity
✔ visualizes kidney stress state
✔ simulates future deterioration or recovery
✔ recommends treatment adjustments
✔ provides AI-based medical explanations
✔ generates downloadable doctor reports

# 🔬 Working Flow of the System

1. User registers/logs in
2. Patient enters kidney-related medical parameters
3. Backend prediction engine processes health risk
4. Kidney simulation module computes damage/stress progression
5. AI engine generates medical reasoning
6. Frontend dashboard displays graphs + kidney visualization
7. Doctor report is generated for download

# ▶️ How to Run the Project Locally

## Clone Repository

```bash
git clone https://github.com/Amrutha-Pawar/Digital-twin_for_kidney_simulation.git
```

## Move into Project

```bash
cd Digital-twin_for_kidney_simulation
```

## Install Backend Requirements

```bash
pip install -r backend/requirements.txt
```

## Run Flask Server

```bash
python backend/app.py
```

## Open Frontend

Open `frontend/index.html` in browser.
